## 개요

### OneDayClass+ 소개
강사들이 하루체험 할 수 있는 공방들을 등록하여 고객들이 예약 할 수 있는 서비스웹사이트

### 주요기능
* 강사는 교육일정과 클래스내용을 등록가능
* 일반회원은 VIP이용권 구매와 클래스예약가능
* 관리자는 회원목록과 강사들이 등록한 클래스를 볼 수 있습니다.

## 회원가입 및 로그인
* 강사와 일반 회원가입할 수 있도록 구현
* 로그인창에서 관리자 / 일반 / 강사로 로그인할 수 있도록 구현
